      PANDUAN TESTING APLIKASI TIMEFOOD
=========================================================

1. CARA MENJALANKAN APLIKASI (Di Komputer Lokal)
---------------------------------------------------------
Pastikan XAMPP/Laragon sudah menyala (Apache & MySQL).
Buka terminal/CMD di dalam folder project ini, lalu jalankan 2 perintah berikut di tab/jendela terminal yang berbeda:

Terminal 1:
php artisan serve

Terminal 2:
npm run dev

Buka browser dan akses: http://127.0.0.1:8000


2. RESET DATABASE & DATA DUMMY
---------------------------------------------------------
Jika kamu baru pertama kali menjalankan aplikasi ini, atau ingin mengembalikan data ke keadaan awal (reset), jalankan perintah ini di terminal:

php artisan migrate:fresh --seed

(Perintah ini akan membuat ulang database dan mengisi data makanan beserta gambar dummy otomatis).
